# Admin App

React + Vite + TanStack Router (file-based) + TanStack Query + ORPC. Shares the same env + ORPC wiring as the user app and consumes `@template/contracts`.

## Scripts

- `pnpm dev` — start Vite dev server on port 5175
- `pnpm build` — Vite build + `tsc` typecheck
- `pnpm test` — Vitest (jsdom) with MSW handlers
- `pnpm lint:check` / `pnpm format:check` — oxlint + Prettier

## Environment

- Copy `.env.example` to `.env`. Parsed via `getPublicEnv()` in `src/env.ts`.
- `VITE_RPC_URL` — backend base URL (default `http://localhost:3000`)
- `VITE_ORPC_VALIDATE_REQUESTS` — enables client-side ORPC request validation (defaults to `true`).

## Routing & API

- File-based routes in `src/routes/**` generated by TanStack Router plugin.
- ORPC client lives in `src/api/orpc-client.ts`, using contracts from `@template/contracts`; validation toggled via env.

## Testing

- Vitest + jsdom with MSW handlers in `src/mocks/handlers.ts`. Tests live next to source files.
