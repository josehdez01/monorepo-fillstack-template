# Landing Page

Marketing/front-door app built with React + Vite + TanStack Router (file-based) + TanStack Query + ORPC client for dynamic data.

## Scripts

- `pnpm dev` — start Vite dev server on port 5173
- `pnpm build` — Vite build + `tsc` typecheck
- `pnpm test` — Vitest (jsdom) with MSW handlers
- `pnpm lint:check` / `pnpm format:check` — oxlint + Prettier

## Environment

- Copy `.env.example` to `.env`. Parsed via `getPublicEnv()` in `src/env.ts`.
- `VITE_RPC_URL` — backend base URL (default `http://localhost:3000`)
- `VITE_ORPC_VALIDATE_REQUESTS` — enables client-side ORPC request validation (defaults to `true`).

## Routing & API

- File-based routes live in `src/routes/**` (generated by TanStack Router plugin).
- ORPC client in `src/api/orpc-client.ts` consumes shared contracts from `@template/contracts`; request validation toggled via env.

## Testing

- Vitest + jsdom. MSW handlers in `src/mocks/handlers.ts`. Tests colocated with source files.
