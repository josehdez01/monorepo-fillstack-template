# User App

React + Vite + TanStack Router (file-based) + TanStack Query + ORPC. Uses `@template/ui/retroui` for UI and `@template/env` for runtime env validation.

## Scripts

- `pnpm dev` — start Vite dev server on port 5175
- `pnpm build` — Vite build + `tsc` typecheck
- `pnpm test` — Vitest (jsdom) with MSW handlers
- `pnpm lint:check` / `pnpm format:check` — oxlint + Prettier

## Environment

- Copy `.env.example` to `.env` (or set via shell). Variables are parsed via `getPublicEnv()` in `src/env.ts`.
- `VITE_RPC_URL` — backend base URL (default `http://localhost:3000`)
- `VITE_ORPC_VALIDATE_REQUESTS` — enables client-side ORPC request validation (defaults to `true`).

## Routing & API

- File-based routes live in `src/routes/**` (generated by TanStack Router plugin).
- ORPC client is in `src/api/orpc-client.ts` and uses contracts from `@template/contracts`. Request validation is toggled by env.

## Testing

- Vitest + jsdom. MSW handlers in `src/mocks/handlers.ts`. Tests live alongside source files (`*.test.ts(x)`).
